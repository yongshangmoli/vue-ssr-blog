# vue-ssr-demo

> 基于`webpack4`构建的`vue`服务端渲染模板，服务端渲染主要是为了优化`seo`及首屏渲染

### 运行

```
# development
npm run dev

# production
npm run build
npm start
```

### 特性

- `eslint`规范代码
- `store`注入解决`cookie`共享
- `commit`时使用`lint-staged`+`husky·对修改部分规范化
- 支持`stylus`
